# 🛡 Password Strength Checker

A cybersecurity project that evaluates password strength using length checks, 
character variety analysis, and a leaked-password database lookup.

**Built for:** DecodeLabs Industrial Training Kit — Project 1, Batch 2026

## Features
- Real-time password strength evaluation (Weak / Medium / Strong)
- Live requirements checklist (length, uppercase, lowercase, numbers, symbols)
- Detection against common leaked passwords
- Two interfaces: Console (CLI) and GUI (Tkinter)
- Dark-themed modern UI

## How to Run

Requires Python 3.9 or higher.

### Console Version
```bash
python3 password_checker.py
